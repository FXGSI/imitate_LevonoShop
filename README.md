# imitate_LevonoShop
联想商城(仿)
-----

一个用JQuery,nodeJs,模仿的联想商城首页,产品搜索页,产品详情页以及购物车页面
实现功能包括：
      主页轮播图， 用户登录，搜索框搜索产品，产品详情页
